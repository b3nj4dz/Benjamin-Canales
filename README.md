# paso 0
1.- Descargar, instalar git
2.- Descargar, istalar VSC
3.- Crear un directorio y abrir con VSC
4.- Abrir la terminal y probar los comandos:
    git -v ==> para ver la versión de git
    git config --global user.name "nombre" ==> agregar nombre de usuario
    git config --global user.email "correo" ==> agregar correo del usuario
5.- Clonar el repositorio creado con Github:
    git clone http:\\...git ==> url del repositorio
# Paso 1
Cada cambio que hacemos dentro del proyecto local cuando queremos guardar
debemos hacer lo siguiente:
    git add. ==> prepara los archivos
    git commit -m "comentario del avanze" ==> crea un punto de guardado
    git push origin main ==> sube los cambios al repositorio en la nube
